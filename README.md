# Vue.js + TypeScript + Parcel boilerplate

This is a very simple boilerplate with Vue.js, TypeScript, Parcel.

## Usage

```bash
# install dependencies
$ yarn install

# start development server
$ yarn serve

# build for production
$ yarn build

# lint
$ yarn lint

# run tests
$ yarn test
```
